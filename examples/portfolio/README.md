# Bench-ready hypothesis portfolio

A growing set of **cited, re-verifiable** oncology hypotheses — public ChEMBL bioactivity triaged and
prioritized, each with an **experimental-validation request** (`VALIDATION-REQUEST.md`) listing the
exact experiments and the real labs/CROs (and free NCI-60 / SGC routes) that would test it.

These are **hypotheses for the bench, not validated hits.** Every figure is fetched-from-source or
deterministically computed and re-checkable: `python3 cad/verify.py --run examples/portfolio/<target>`.
Generated by `cad/run_pipeline.py` + `cad/validation_package.py`. Research only; not medical advice.
