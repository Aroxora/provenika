# Provenika — CADD pipeline Python dependencies
# =============================================================================
# The cad/ and cicd/ tools are STANDARD-LIBRARY ONLY except for RDKit, which is
# needed by just three modules (cheminformatics.py, virtual_triage.py --query,
# precompute_site_data.py). All of them degrade gracefully — if RDKit is absent
# they print `pip install rdkit` and exit cleanly (code 3); they never fake a
# result. So the *minimum* to run target triage, structure fetch, the docking
# box, cost-benefit, the full pipeline, and verify.py is: nothing but Python.
#
# Install everything below with:  pip install -r cad/requirements.txt
# Python floor: 3.10+ (the code runs on 3.9+ thanks to `from __future__ import
# annotations`, but 3.10 is the supported/tested floor; CI uses 3.12).
# =============================================================================

rdkit>=2024.3        # cheminformatics: descriptors, QED, PAINS/Brenk, scaffolds,
                     # ECFP4 Tanimoto similarity. Wheel name is `rdkit` (NOT the
                     # old `rdkit-pypi`). Without it the pipeline still runs;
                     # ChEMBL-computed properties are used and similarity is off.

# -----------------------------------------------------------------------------
# DOCKING (optional stage 5, cad/dock.py + cad/validate.py --redock). Two layers:
#
#  1. Docking-grade PREP — pip-installable, and what produces the validated ~1.4 Å redocking
#     median (vs ~7.9 Å on the Open-Babel-only fallback). Install with:
#         pip install -r cad/requirements-docking.txt        # Meeko + pdb2pqr (needs the RDKit above)
#
#  2. The Vina + Open Babel BINARIES — NOT pip-installable on recent Python (`pip install vina`
#     has no wheels / needs Boost; no Homebrew `vina` formula). Use conda:
#         conda install -c conda-forge vina openbabel        # recommended (all OSes)
#         # macOS without conda: brew install open-babel, then a Vina release binary on PATH
#         #   from https://github.com/ccsb-scripps/AutoDock-Vina/releases
#
# Check the WHOLE stack (binaries + prep):  python3 cad/dock.py --check
# Without the prep layer dock.py still runs, on the inferior Open-Babel path — never a fake score.
# -----------------------------------------------------------------------------
